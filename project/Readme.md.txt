Please follow the below steps to run the application

1.Start mongodb
2.Navigate to server directory in node command prompt and type 'npm install'
3.Now start node using the command 'node server.js'
4.Open one more node command prompt and navigate to src directory and type 'npm install'
5.Generate the build using the command 'gulp'
6.Finally start the server using command 'gulp webserver'
7.Open the below url in the browser


********http://localhost:8080/#/login********
